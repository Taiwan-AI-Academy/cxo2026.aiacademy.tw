- **活動地點**：台北萬豪酒店五樓萬豪廳（台北市中山區樂群二路199號）

## 資源檔案規格

### Logo 檔案
- **檔案位置**：`assets/logo_capture.svg`
- **要求**：透明背景（已移除藍色背景）
- **使用位置**：導航列 Logo 區域
- **建議尺寸**：寬度 32-40px，高度自動調整

### 講師照片
- **檔案命名**：`assets/1-chen.jpeg`、`assets/2-lien.jpg`、`assets/3-yu.jpg`、`assets/4-lee.jpg`
- **顯示尺寸**：
  - 行動裝置：`w-40 h-40`（160px × 160px）
  - 桌面裝置：`w-48 h-48`（192px × 192px）
- **樣式**：圓形、金色邊框、陰影效果

### 社群夥伴 ICON 規格

#### 最佳尺寸建議
- **原始檔案尺寸**：256px × 256px（2x 解析度）或 512px × 512px（高解析度）
- **顯示尺寸**：128px × 128px（`w-32 h-32`）
- **檔案格式**：PNG（透明背景）或 SVG（向量圖）
- **色彩模式**：RGB
- **背景**：透明（適用於深色背景）

#### 各夥伴 ICON 建議
1. **Google**：使用官方 Logo，白色或彩色版本（視背景調整）
2. **Microsoft**：使用官方 Logo，白色或彩色版本
3. **AWS**：使用官方 Logo，白色或彩色版本
4. **NVIDIA**：使用官方 Logo，白色或彩色版本
5. **IBM**：使用官方 Logo，白色或彩色版本

#### 技術規格
- **寬高比**：1:1（正方形）
- **內邊距**：建議保留 10-15% 的內邊距，避免圖示貼邊
- **檔案大小**：建議 < 50KB（PNG）或 < 20KB（SVG）
- **命名規範**：`partner-google.png`、`partner-microsoft.png` 等

## 開發與維護指南

### 本地開發
```bash
# 安裝依賴
npm install

# 啟動本地伺服器
npm run serve
# 或使用其他靜態伺服器
```

### 修改建議

#### 更新 Logo
1. 確保 SVG 檔案為透明背景
2. 在導航列區塊（約第 163-165 行）替換現有 Logo 元素
3. 使用 `<img>` 標籤引入 SVG 檔案

#### 更新社群夥伴 ICON
1. 將 ICON 檔案放置於 `assets/` 目錄
2. 在社群夥伴區塊（約第 634-678 行）替換文字為 `<img>` 標籤
3. 使用 `object-contain` 確保圖示完整顯示

#### 更新講師資訊
1. 修改 `class_data.txt` 或直接編輯 HTML
2. 確保講師照片檔案存在於 `assets/` 目錄
3. 更新講師區塊的內容與照片路徑

### 瀏覽器相容性
- **現代瀏覽器**：Chrome、Firefox、Safari、Edge（最新版本）
- **CSS 特性**：使用 `backdrop-filter`，需確認瀏覽器支援
- **響應式設計**：支援行動裝置與桌面裝置
